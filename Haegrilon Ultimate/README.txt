Installation: 
══════════════════════════════════════════════════════════════════════════
> Launch Minecraft and click on "Options"

> Click on "Resource Packs"

> Click on "Open Resource Pack Folder"

> Drag the Excalibur.zip file into the resource pack folder
  Alternatively you can click on "Windows Start" > Run and type: %appdata%/.minecraft/resourcepacks
  
> Select the Excalibur resource pack from the left column and move it to the top of the "Selected Resource Packs" column

> Select "Done" the game will now start to load the textures

> Enter your adventure
